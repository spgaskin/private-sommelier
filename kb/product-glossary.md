# Product Glossary

- **Atlas**: the internal planning and roadmap system. Source of truth for initiative status.
- **Tunnel**: the SSH tunnel that is the only network path into the private box; nothing else is exposed.
- **Judger**: a verification pass that checks an agent's output before it is trusted or acted on.
- **Skill**: a reusable procedure the agent banks from experience so it does not re-derive brittle steps each run.
- **Sovereignty tier**: low tier = local-only research/drafting/code; high tier = network-reachable, multi-user, or holds production credentials (needs formal review).
